Example in Python¶

Here is for instance how we can compute forward dynamics for a manipulator using the CRBA and RNEA implementations of the Pinocchio library:

from robot_descriptions.loaders.pinocchio import load_robot_description

def forward_dynamics_crba_rnea(model, data, q, v, tau):
    # Inertia matrix with CRBA
    M = pin.crba(model, data, q)

    # Bias torques with RNEA
    zero_accel = np.zeros(model.nv)
    tau_0 = pin.rnea(model, data, q, v, zero_accel)

    # Forward dynamics by solving the linear system
    a = np.linalg.solve(M, tau - tau_0)
    return a
